A next-generation mod manager built with [Svelte 5](https://svelte.dev/), [Tauri 2](https://tauri.app/) and [Rust](https://www.rust-lang.org/). Forked from [Gale](https://github.com/Kesomannen/gale).

## Features

- Support for all 150+ games on Thunderstore, including Lethal Company, R.E.P.O and Risk Of Rain 2
- An intuitive and responsive dark interface
- Tiny download size and resource usage
- Feature-rich mod config editor
- Automatic profile syncing (beta)
- Multi-source support (coming soon)
- Social features and community profiles (coming soon)

## Installation

### Windows

- Go to [Releases](https://github.com/prismo-studio/zephyr/releases)
- Download the `.msi` installer for your desired version (latest recommended)
- Run the downloaded file

> [!NOTE]
> You might get a prompt saying "Windows has protected your PC". In this case, click `More Info` and `Run Anyway`.

### Linux

AppImages, .rpm and .deb files are available in [Releases](https://github.com/prismo-studio/zephyr/releases).
